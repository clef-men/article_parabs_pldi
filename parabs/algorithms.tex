\section{Parallel Iterators}
\label{sec:parabs_algorithms}

On top of \ocamlinline{Future}, we  implemented and verified standard parallel iterators~\refLibTheoriesFile{zoo_parabs/algo} that are particularly useful for benchmarks (see \cref{sec:benchmarks}): \c{for\_}, \c{for\_each}, \c{fold} and \c{find}. For reasons of space we do not detail all of them, but for example \ocamlinline{Algo.for_} has the following type
\begin{ocamlcode}
val for_ :
  Pool.context ->
  beg:int (* begin index (included) *) ->
  end_:int (* end index (excluded) *) ->
  chunk:int option (* recommended sequential cutoff size *) ->
  (int -> int -> unit) Pool.task -> unit
\end{ocamlcode}
%
and could be used as follows:
\begin{ocamlcode}
let double_array ctx t =
  Algo.for_ ctx ~beg:0 ~end_:(Array.length t) ~chunk:None 
    (fun _ctx begin_ end_ ->
      for i = begin_ to end_ - 1 do
        t.(i) <- 2 * t.(i)
      done)
\end{ocamlcode}
